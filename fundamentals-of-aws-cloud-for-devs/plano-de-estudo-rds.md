## Passos do plano em si

1. Motivação:
  
Este plano de estudos tem como objetivo fornecer aos estudantes os conhecimentos e habilidades necessárias para entender e utilizar o serviço Relational Database Service (RDS) da Amazon Web Services (AWS). Aprender a usar esse serviço é importante para desenvolver habilidades em implantação e gerenciamento de bancos de dados relacionais na nuvem da AWS.

2. Objetivo de aprendizagem:

Ao finalizar este plano de estudos, o estudante será capaz de criar e gerenciar instâncias de banco de dados usando o serviço RDS da AWS, bem como implantar aplicações que utilizam esses bancos de dados na nuvem da AWS.

3. Pré-requisitos:

Para seguir este plano de estudos, o estudante deve ter conhecimentos básicos em programação, sistemas operacionais, redes de computadores e em bancos de dados relacionais.

4. Objetivos de aprendizagem conectados com o nível 3 da taxonomia de bloom:

- Compreender o conceito de instâncias de banco de dados e sua relação com serviços em nuvem.
- Configurar o ambiente AWS para uso do serviço RDS.
- Criar e gerenciar instâncias de banco de dados usando o RDS.
- Implantar aplicações que utilizam bancos de dados relacionais na nuvem da AWS usando o RDS.

5. Lista de temas que serão abordados no plano, com referências e objetivos de aprendizagem específicos:

- Introdução ao RDS
  - Objetivo de aprendizagem: Compreender o conceito de instâncias de banco de dados e sua relação com serviços em nuvem.
  - Referências: "Amazon RDS User Guide" da AWS, Capítulo 1.

- Configuração do Ambiente AWS para Uso do RDS
  - Objetivo de aprendizagem: Configurar o ambiente AWS para uso do serviço RDS.
  - Referências: "Amazon RDS User Guide" da AWS, Capítulo 2.

- Criação e Gerenciamento de Instâncias de Banco de Dados
  - Objetivo de aprendizagem: Criar e gerenciar instâncias de banco de dados usando o RDS.
  - Referências: "Amazon RDS User Guide" da AWS, Capítulo 3.

- Implantação de Aplicações que Utilizam Bancos de Dados Relacionais na Nuvem da AWS
  - Objetivo de aprendizagem: Implantar aplicações que utilizam bancos de dados relacionais na nuvem da AWS usando o RDS.
  - Referências: "Amazon RDS User Guide" da AWS, Capítulo 4.

6. Palavras-chave:

RDS, instâncias de banco de dados, serviços em nuvem, implantação de aplicações.

7. Exercícios:

- Crie uma instância de banco de dados usando o serviço RDS da AWS.
- Crie uma tabela no banco de dados criado anteriormente e insira alguns registros.
- Crie uma aplicação web em Flask que utiliza a tabela criada anteriormente e faça o deploy dessa aplicação na nuvem da AWS usando o serviço Elastic Beanstalk da AWS.
- Configure a conexão da aplicação web criada anteriormente com a instância de banco de dados criada no passo 1.
- Teste a aplicação web implantada na nuvem da AWS para verificar se a conexão com o banco de dados está funcionando corretamente.
